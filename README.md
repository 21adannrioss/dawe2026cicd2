# dawe2026cicd2
Pràctica RA6 CICD intro

## Adan El Ouadah De Rios
L'aplicació està estructurada en tres seccions principals dins de l'HTML: una capçalera amb el títol, una secció per a afegir noves tasques mitjançant un formulari i dues llistes separades per a tasques pendents i completades.
El CSS està organitzat amb variables globals en :root per a facilitar la personalització de colors i estils, i la resta del codi es divideix per components (capçalera, formulari, llistes i responsive).
El JavaScript gestiona la lògica de l'aplicació, permetent carregar, guardar i actualitzar tasques en localStorage, a més d'afegir noves tasques i moure-les entre pendents i completades en fer clic.
